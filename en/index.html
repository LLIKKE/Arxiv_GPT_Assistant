### Daily Research Summary

Date: October 26, 2023

Today's publications highlight significant advancements across several key research vectors, with a strong emphasis on refining the training and operational paradigms of large models. The work demonstrates a clear trend toward creating more dynamic, efficient, and contextually aware AI systems.

Agents & Long-Context Modeling

Research continues to push the boundaries of agentic systems toward true long-term personalization. A key breakthrough is the development of rigorous benchmarks for evaluating an agent's ability to maintain and utilize lifelong, cross-domain user memory. This moves beyond simple context window extensions, focusing instead on the core challenge of building self-evolving agents that can leverage historical interactions for personalized behavior, a critical step toward practical, persistent AI assistants.

RL for LLMs & Core Model Optimization

Reinforcement Learning is being applied to increasingly fundamental components of the model architecture. Today's work showcases this through a novel RL framework that dynamically optimizes tokenization—the very process of converting text to model inputs. This "reinforcement patching" approach allows for end-to-end training and demonstrates zero-shot transfer, suggesting RL can be used not just for aligning outputs (RLHF) but for co-designing and improving core, pre-training-level model efficiencies.

Data-Centric Training & SFT

The paradigm of data-centric AI is being formally unified for large language models. New frameworks are emerging that treat the training dataset not as a static corpus but as a dynamic, optimizable component. These systems introduce novel strategies for intelligent data sourcing, curation, and scheduling throughout the SFT and continued pre-training phases, aiming to maximize learning efficiency and model capability by algorithmically managing the data pipeline.

Multimodal & Structured Reasoning

In the multimodal domain, there is a concerted effort to impose more rigorous, structured reasoning processes on complex inputs like video. The latest approaches combine Chain-of-Thought prompting with reinforcement learning to train MLLMs. This RL framework specifically rewards the model for generating and following logically structured reasoning traces when answering video-based questions, leading to more interpretable and robust video understanding by enforcing disciplined, step-by-step analysis.

1. MemoryCD: Benchmarking Long-Context User Memory of LLM Agents for Lifelong Cross-Domain Personalization

ArXiv ID: 2603.25973 |

👥 Authors: Weizhi Zhang, Xiaokai Wei, Wei-Chieh Huang, Zheng Hui, Chen Wang, Michelle Gong, Philip S. Yu

🏫 Affiliations: Unknown Institution

💡 Highlight: This paper introduces a benchmark for evaluating long-context memory in LLM agents, directly relevant to building self-evolving agents that require lifelong user memory for personalization.
View Abstract

arXiv:2603.25973v1 Announce Type: new Abstract: Recent advancements in Large Language Models (LLMs) have expanded context windows to million-token scales, yet benchmarks for evaluating memory remain limited to short-session synthetic dialogues. We introduce \textsc{MemoryCD}, the first large-scale, user-centric, cross-domain memory benchmark derived from lifelong real-world behaviors in the Amazon Review dataset. Unlike existing memory datasets that rely on scripted personas to generate synthetic user data, \textsc{MemoryCD} tracks authentic user interactions across years and multiple domains. We construct a multi-faceted long-context memory evaluation pipeline of 14 state-of-the-art LLM base models with 6 memory method baselines on 4 distinct personalization tasks over 12 diverse domains to evaluate an agent's ability to simulate real user behaviors in both single and cross-domain settings. Our analysis reveals that existing memory methods are far from user satisfaction in various domains, offering the first testbed for cross-domain life-long personalization evaluation.

2. Dynamic Tokenization via Reinforcement Patching: End-to-end Training and Zero-shot Transfer

ArXiv ID: 2603.26097 |

👥 Authors: Yulun Wu, Sravan Kumar Ankireddy, Samuel Sharpe, Nikita Seleznev, Dehao Yuan, Hyeji Kim, Nam H. Nguyen

🏫 Affiliations: Unknown Institution

💡 Highlight: This paper proposes a reinforcement learning framework for dynamic tokenization, which is relevant to RL for LLMs as it applies RL to optimize a core component of sequence modeling.
View Abstract

arXiv:2603.26097v1 Announce Type: new Abstract: Efficiently aggregating spatial or temporal horizons to acquire compact representations has become a unifying principle in modern deep learning models, yet learning data-adaptive representations for long-horizon sequence data, especially continuous sequences like time series, remains an open challenge. While fixed-size patching has improved scalability and performance, discovering variable-sized, data-driven patches end-to-end often forces models to rely on soft discretization, specific backbones, or heuristic rules. In this work, we propose Reinforcement Patching (ReinPatch), the first framework to jointly optimize a sequence patching policy and its downstream sequence backbone model using reinforcement learning. By formulating patch boundary placement as a discrete decision process optimized via Group Relative Policy Gradient (GRPG), ReinPatch bypasses the need for continuous relaxations and performs dynamic patching policy optimization in a natural manner. Moreover, our method allows strict enforcement of a desired compression rate, freeing the downstream backbone to scale efficiently, and naturally supports multi-level hierarchical modeling. We evaluate ReinPatch on time-series forecasting datasets, where it demonstrates compelling performance compared to state-of-the-art data-driven patching strategies. Furthermore, our detached design allows the patching module to be extracted as a standalone foundation patcher, providing the community with visual and empirical insights into the segmentation behaviors preferred by a purely performance-driven neural patching strategy.

3. DataFlex: A Unified Framework for Data-Centric Dynamic Training of Large Language Models

💡 Highlight: This paper presents a unified framework for data-centric dynamic training of LLMs, directly addressing novel data-sourcing and optimization strategies for SFT and pretraining.
View Abstract

arXiv:2603.26164v1 Announce Type: new Abstract: Data-centric training has emerged as a promising direction for improving large language models (LLMs) by optimizing not only model parameters but also the selection, composition, and weighting of training data during optimization. However, existing approaches to data selection, data mixture optimization, and data reweighting are often developed in isolated codebases with inconsistent interfaces, hindering reproducibility, fair comparison, and practical integration. In this paper, we present DataFlex, a unified data-centric dynamic training framework built upon LLaMA-Factory. DataFlex supports three major paradigms of dynamic data optimization: sample selection, domain mixture adjustment, and sample reweighting, while remaining fully compatible with the original training workflow. It provides extensible trainer abstractions and modular components, enabling a drop-in replacement for standard LLM training, and unifies key model-dependent operations such as embedding extraction, inference, and gradient computation, with support for large-scale settings including DeepSpeed ZeRO-3. We conduct comprehensive experiments across multiple data-centric methods. Dynamic data selection consistently outperforms static full-data training on MMLU across both Mistral-7B and Llama-3.2-3B. For data mixture, DoReMi and ODM improve both MMLU accuracy and corpus-level perplexity over default proportions when pretraining Qwen2.5-1.5B on SlimPajama at 6B and 30B token scales. DataFlex also achieves consistent runtime improvements over original implementations. These results demonstrate that DataFlex provides an effective, efficient, and reproducible infrastructure for data-centric dynamic training of LLMs.

4. Reinforcing Structured Chain-of-Thought for Video Understanding

ArXiv ID: 2603.25942 |

👥 Authors: Peiyao Wang, Haotian Xu, Noranart Vesdapunt, Rui Hou, Jingyi Zhang, Haibin Ling, Oleksandr Obiednikov, Ning Zhou, Kah Kuen Fu

🏫 Affiliations: Unknown Institution

💡 Highlight: This paper introduces a novel RL framework for video understanding in MLLMs, combining interests in RL for LLMs, Multimodal LLMs, and advanced fine-tuning strategies.
View Abstract

arXiv:2603.25942v1 Announce Type: new Abstract: Multi-modal Large Language Models (MLLMs) show promise in video understanding. However, their reasoning often suffers from thinking drift and weak temporal comprehension, even when enhanced by Reinforcement Learning (RL) techniques like Group Relative Policy Optimization (GRPO). Moreover, existing RL methods usually depend on Supervised Fine-Tuning (SFT), which requires costly Chain-of-Thought (CoT) annotation and multi-stage training, and enforces fixed reasoning paths, limiting MLLMs' ability to generalize and potentially inducing bias. To overcome these limitations, we introduce Summary-Driven Reinforcement Learning (SDRL), a novel single-stage RL framework that obviates the need for SFT by utilizing a Structured CoT format: Summarize -> Think -> Answer. SDRL introduces two self-supervised mechanisms integrated into the GRPO objective: 1) Consistency of Vision Knowledge (CVK) enforces factual grounding by reducing KL divergence among generated summaries; and 2) Dynamic Variety of Reasoning (DVR) promotes exploration by dynamically modulating thinking diversity based on group accuracy. This novel integration effectively balances alignment and exploration, supervising both the final answer and the reasoning process. Our method achieves state-of-the-art performance on seven public VideoQA datasets.

Research Interest Criteria

Please evaluate the papers based on the following three core areas:

1. Supervised Fine-Tuning (SFT) for LLMs

Supervised Fine-Tuning (SFT) is a critical technique for adapting pre-trained Large Language Models (LLMs) to specific downstream tasks or domains. This process involves further training the model on a curated dataset of high-quality, labeled examples that reflect the target behavior. Our research focuses on optimizing the SFT process to enhance model performance, alignment, and efficiency.

  • Key Direction 1: Investigating novel data-sourcing and data-synthesis strategies to construct high-quality, diverse, and instruction-rich SFT datasets.
  • Key Direction 2: Developing advanced SFT algorithms and optimization techniques to improve the efficiency and effectiveness of fine-tuning on specialized tasks.
  • Potential Contribution: Establishing best practices for curating SFT data and designing fine-tuning protocols that lead to more capable and reliable LLMs.

2. Reinforcement Learning (RL) for LLMs

Reinforcement Learning, particularly from human or AI-generated feedback, offers a powerful paradigm for aligning LLM behavior with complex human preferences and objectives. This research area explores methods beyond simple instruction-following, aiming to imbue models with more nuanced capabilities. Our work centers on creating robust RL pipelines and novel reward mechanisms to steer LLM development.

  • Relevance: We explore cutting-edge RL techniques, including rejection sampling and unsupervised reinforcement learning with verifiable rewards, to build sophisticated data pipelines that continuously enhance LLM capabilities.
  • Key Direction: Designing novel reward functions that directly target improvements in key metrics such as helpfulness, honesty, harmlessness, and reductions in hallucination.
  • Potential Contribution: Pioneering new research directions in RL for LLMs that inspire further investigation and bridge the gap between theoretical advancements and practical, large-scale model improvements.

3. Agents for LLMs

Autonomous agents built upon LLMs are a key step towards creating general-purpose problem-solving systems. Our emphasis is on self-evolving agents: agents that can iteratively improve their policies, tools, and internal knowledge through experience. This direction studies the mechanisms that make agent behavior not only autonomous at inference time, but continually improving over long horizons via self-generated feedback and data. The goal is to design agent architectures that translate self-evolution into measurable capability gains and strong real-world deployability.

  • Relevance: Centering on self-evolving agents—agent systems that autonomously generate supervision signals, curate experience, and improve themselves over time rather than remaining static after deployment.
  • Key Direction: Building closed-loop improvement frameworks, e.g., self-reflection and self-critique, automated curriculum generation, and learning from interaction traces (including tool-use trajectories) to drive continual adaptation.
  • Potential Contribution: Delivering practical, deployable self-evolving agents that demonstrate sustained improvements in reasoning, planning, and tool use, with clear evaluation protocols that quantify long-horizon learning progress and reliability.
