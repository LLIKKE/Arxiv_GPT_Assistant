💡 今日研究速览 (Daily Summary)

SFT & Distillation

A significant trend in post-training is the convergence of on-policy optimization and knowledge distillation. Several works propose new algorithms that move beyond naive KL divergence by using teacher models to provide richer, denser supervisory signals. TGPOintroduces teacher-conditioned directional guidance to address distribution mismatch, whileEffOPDprovides a plug-and-play acceleration method for the on-policy distillation paradigm, offering novel parameter-level analysis of efficiency mechanisms.MOPDtakes this further by leveraging peer successes and failures from multiple rollouts to construct denser teacher signals, moving away from a single static teacher. Meanwhile,RESDtackles the challenging rare-success regime by transforming failure feedback into corrective supervision via a self-distillation loop, demonstrating that effective learning can occur even when positive examples are scarce.EGRSDadds another dimension by using teacher-entropy confidence gating to improve the accuracy-length frontier in reasoning tasks. Collectively, these works indicate a maturation of distillation techniques that are tightly integrated with on-policy RL dynamics.

RL for LLMs (Reasoning & Alignment)

The field of RL for LLMs is experiencing a wave of sophisticated refinements to foundational algorithms like GRPO and PPO, moving beyond basic reward design to address structural and statistical pathologies.ConSPOreframes RLVR from a contrastive perspective to overcome the structural limitations of GRPO.Taming Extreme Tokensintroduces a covariance-aware, hyperparameter-free improvement to GRPO by reweighting advantages based on token-level variance.ODRPOtackles the critical issue of noisy auto-rater feedback by decomposing discrete rewards into ordinal binary indicators for robust optimization.RDPOaddresses multi-objective and mixed-reward settings by decorrelating rewards to stabilize advantage estimation.YFPOexplores a novel direction by using neuron-level activation signals as an auxiliary reward for mathematical reasoning.GRACEoffers a complementary approach by providing a gradient-aligned method for curating high-quality reasoning data for efficient post-training. Finally,SLOPaddresses the persistent problem of reward hacking at inference time by using a sharpened logarithmic opinion pool of generative reward models. These papers collectively signal a shift from "bigger is better" reward models to more principled, stable, and interpretable optimization frameworks.

Agents (Self-Improving & Multi-Agent Systems)

The dominant theme in agent research is the creation of closed-loop, self-evolving systems that improve without direct human intervention, often through a co-evolutionary or dialectic process.EvoGroundandAgent-BRACEboth propose frameworks where agents improve via RL, with the former focusing on video grounding and the latter on decoupling belief from policy for long-horizon tasks.Harnessing Agentic Evolutionintroduces a meta-editing framework that uses accumulated evolution context to steer long-horizon improvement.SkillGraphcreates a self-evolving skill graph from trajectories and RL feedback, enabling compositional task improvement.MANGOextends this to multi-agent systems, using RL and textual gradients to jointly optimize workflow paths and agent behaviors.CHALproposes a novel multi-agent dialectic framework for belief optimization over defeasible domains, introducing gradient-informed belief revision. A critical counterpoint is provided byUseful Memories Become Faulty, which reveals that LLM-based consolidated memory degrades over time, advocating for episodic-only memory as a more robust alternative. This highlights a key tension in self-improving systems: the need for memory versus its tendency toward corruption.

RL for Agents (Action Guidance & Trajectory Optimization)

A specific sub-field is emerging that focuses on using RL to train LLM agents by leveraging action-level guidance rather than just outcome-based rewards.Learning Agentic Policy from Action Guidancedirectly addresses the reward reachability barrier by using human data for action guidance, enabling self-improvement without initial SFT.Revisiting DAgger in the Era of LLM-Agentsapplies a classic imitation learning algorithm to mitigate covariate shift in agent trajectories, showing strong results on SWE-bench.Achieving Gold-Medal-Level Olympiad Reasoningpresents a unified RL pipeline that uses both verifiable rewards and proof-level RL, demonstrating that scaling simple RL recipes can produce exceptional reasoning capabilities.Scaling Retrieval-Augmented Reasoningproposes an RL-based framework with parallel search and explicit merging for improved reasoning. TheBicameral Modeloffers a fascinating departure by proposing a latent communication interface between LLMs via hidden-state coupling, enabling tool use without text serialization, which could fundamentally change how agents are architected. These works signal a move toward more structured and guided RL training for agentic behavior.

Multimodal & Robotics

Multimodal and robotics research is increasingly adopting RL fine-tuning to achieve robust and task-aligned behavior.What to Ignore, What to Reactproposes a visually robust RL fine-tuning framework for VLA models that explicitly separates invariance and sensitivity objectives, directly addressing a key weakness of current models.GRIP-VLMuses GRPO as a discrete optimization framework for pruning visual tokens, demonstrating that RL can be a powerful tool for efficiency.RoboEvolveintroduces a co-evolutionary loop between a VLM planner and a VGM simulator, creating a self-improving system for robotic manipulation with limited data.VideoSEALaddresses evidence misalignment in long-video understanding with a decoupled planner-inspector framework and novel reward design.CRAFTapplies reward-based finetuning to the domain of medical image synthesis, demonstrating the broader applicability of these techniques.Gaze Attentionintroduces a latent mechanism to selectively attend to compact visual regions, achieving a 90% reduction in KV cache usage, which is a significant practical contribution.

Infrastructure & Tooling

As RL for LLMs becomes a standard post-training paradigm, the need for scalable and efficient infrastructure is being addressed.MinTproposes a managed infrastructure system for the training and serving of millions of LoRA-based LLM policies, directly tackling the scalability challenge of personalized RL post-training.Qwen-Scopeproposes using sparse autoencoders as practical development tools, including for RL-based post-training optimization.TokenRatio provides a principled theoretical framework for token-level preference optimization, recovering optimality from sequence-level preferences. These works indicate a growing maturity in the ecosystem, with a focus on making advanced RL techniques deployable and interpretable at scale.
