💡 Daily Research Summary

### RL for LLMs & Reward Design The dominant theme today is the maturation of Reinforcement Learning for Large Language Models, moving beyond simple outcome-based rewards. A significant cluster of papers addresses the critical challenge of **reward specification and credit assignment**, particularly for open-ended and non-verifiable domains. Works like**Prompt-Level Reward Specifications**and**EvoRubric**propose dynamic, self-evolving rubric generation to replace static, hand-crafted criteria, eliminating the need for expensive human annotations or separate reward models. Similarly,**CorVer**and**Label-Free RL via Cross-Model Entropy**pioneer lightweight, reference-free reward signals, with the latter using cross-model entropy as a universal reward for instruction following. On the training algorithm front, innovations like**HPO**(Hysteretic Policy Optimization) and**ESPO**(Early-Stopping PPO) directly tackle core training instabilities—sparse rewards and wasted compute from failed rollouts—while**RL2ML**provides a family of unbiased gradient estimators for verifiable rewards. The field is also seeing a surge in**process-level supervision**for complex tasks, with**DeepTool**,**GAPD**, and**Beyond Trajectory Rewards**each proposing fine-grained, step-level credit assignment to guide reasoning and tool use. Finally,**REDIPO**and**When RL Suppresses Its Own Vocabulary**highlight a growing awareness of the trade-off between alignment and diversity, proposing novel data-construction and reward-shaping techniques to recover lost behavioral diversity in post-trained models.

Latent Reasoning & Efficient Computation

A second major breakthrough is the push towardefficient, non-autoregressive latent reasoning. This direction seeks to internalize the reasoning process, moving away from verbose, compute-heavy chain-of-thought (CoT) to more compact and efficient internal representations.Unlocking the Working Memory of LLMs (RiM)proposes a radical shift, replacing autoregressive reasoning tokens with fixed memory blocks processed in a single forward pass, dramatically reducing inference cost.Context Distillation as Latent Memory Managementoffers a complementary approach by formulating context as modular, gated LoRA adapters, effectively treating memory as a learnable latent resource.DenseSteerprovides a training-time-free inference method by steering internal representations toward dense reasoning patterns. For more complex planning,Thoughts-as-Planningformalizes CoT optimization as reinforcement learning over a latent world model, enabling planning in this compressed space. TheCosmicFish-HRMmodel introduces a hierarchical architecture that adaptively controls the depth of latent reasoning. Robustness and safety are also being addressed in this paradigm, withCOLAGUARDdemonstrating how multi-step safety reasoning can be distilled into a continuous latent space for efficient, high-throughput guardrailing.

Self-Improving & Tool-Integrated Agents

A key trend is the integration ofprocess-supervised RLto refine agentic behavior at a granular level.GRASP,GrepSeek, andPRO-CUAeach propose frameworks where agents learn to build and edit skill libraries, interact directly with corpora, or perform computer use tasks, guided by step-level rewards from RL. This is complemented by a focus onself-verification and reflection, as seen inSelf-Trained Verification (STV)andBenchTrace, which aim to improve both training-time and test-time self-improvement by enhancing the agent's own ability to critique its actions. TheCurse of Helpfulnesspaper provides a crucial counterpoint, showing that RL (GRPO) can actually restore robustness against distracting instructions, demonstrating a beneficial alignment effect. Furthermore, the boundaries of agentic systems are expanding into new domains:KairosAgentapplies this paradigm to time series forecasting, andLearning Design Skillsto photonic inverse design, both using RL to learn domain-specific policies from feedback. 🏫 Affiliations: Unknown Institution

💡 Highlight: Introduces MMPO, a meta-cognitive memory policy optimization method using belief entropy for fine-grained RL supervision in long-horizon LLM agents, strongly matching both RL for LLMs and self-improving agents.
View Abstract

arXiv:2605.30159v1 Announce Type: new Abstract: Memory-augmented LLM agents tackle complex long-horizon tasks by recursively summarizing interaction trajectories into compact memory. However, existing approaches typically train these memory policies using outcome-based reinforcement learning, failing to localize where intermediate memory quality degrades. As interactions unfold, ambiguous recursive summaries progressively discard task-relevant information and introduce semantic noise. This exacerbates belief deviation, obscuring the agent's estimate of the latent task state and ultimately derailing long-horizon reasoning. We therefore argue that memory optimization should focus not merely on trajectory-level success, but on the clarity of the belief induced by intermediate summaries. To this end, we introduce Belief Entropy, a self-supervised proxy that probes how uncertain the model remains about the latent task state given its current memory. Based on this proxy, we propose Metacognitive Memory Policy Optimization (MMPO). Instead of relying only on sparse outcome-based signals, MMPO provides fine-grained, memory-specific supervision via explicitly penalizing summaries that induce high epistemic uncertainty. Experiments show that MMPO consistently outperforms existing methods on diverse long-horizon tasks, maintaining 97.1% performance even when scaled to 1.75M-token contexts.

3. Label-Free Reinforcement Learning via Cross-Model Entropy

ArXiv ID: 2605.29009 | Relevance Score: 8/10 | Novelty Score: 8/10

👥 Authors: Matt Gorbett, Hossein Shirazi

🏫 Affiliations: Unknown Institution

💡 Highlight: Proposes Cross-Model Entropy (CME), a novel label-free reward signal for RL post-training of LLMs, enabling RL for open-ended instruction following without ground-truth verifiers or human labels.
View Abstract

arXiv:2605.29009v1 Announce Type: new Abstract: Post-training large language models with reinforcement learning is bottlenecked by the reward signal. Existing approaches require either ground-truth verifiable rewards, restricting training to domains with automatic correctness checks (e.g., mathematics, code execution), or human preference labels, which are expensive to collect and prone to reward hacking. Recent label-free methods replace ground-truth verifiers with self-referential signals like majority voting or token entropy over a model's own outputs, but risk reinforcing a model's own errors. In this work we propose Cross-Model Entropy (CME), the mean log-likelihood of a generator's response under a separate verifier model, as a label-free reward signal for RL post-training. CME is continuous, training-free, and grounded in the principle that responses a verifier finds unsurprising are likely correct or high quality. Because the verifier is independent of the generator, the signal cannot be gamed through self-consistency. We integrate CME into GRPO with no other changes to the training loop, extending label-free RL to open-ended instruction following -- a regime where self-referential signals are inapplicable or poorly suited. On open-ended instruction following (UltraFeedback prompts, evaluated on AlpacaEval 2.0), CME rewards beat the untrained base in head-to-head LLM-as-Judge comparisons across four model families (Qwen, Llama, Gemma, OLMo) and three training regimes (pretrained, SFT, and instruction-tuned), with tie-adjusted win rates ranging from 52.5% to 71.4%. Code will be released upon publication.

5. Unlocking the Working Memory of Large Language Models for Latent Reasoning

ArXiv ID: 2605.30343 | Relevance Score: 8/10 | Novelty Score: 8/10

👥 Authors: Lukas Aichberger, Sepp Hochreiter

🏫 Affiliations: Unknown Institution

💡 Highlight: Proposes Reasoning in Memory (RiM), a latent reasoning method that replaces autoregressive generation of reasoning steps with fixed memory blocks processed in a single forward pass, enabling compute-efficient latent reasoning.
View Abstract

arXiv:2605.30343v1 Announce Type: new Abstract: To improve the reasoning capabilities of large language models, test-time compute is typically scaled by generating intermediate tokens before the final answer. However, this couples reasoning to autoregressive generation and thereby conflates internal computation with external communication. In contrast, human cognition can use working memory to hold and manipulate information internally without the need to externalize intermediate thoughts. Drawing on this principle, we introduce Reasoning in Memory (RiM), a latent reasoning method that replaces the autoregressive generation of reasoning steps with memory blocks. These memory blocks are fixed sequences of special tokens that unlock the working-memory capacity of large language models. Since they are fixed rather than generated, they can be processed in a single forward pass, enabling compute-efficient latent reasoning. To operationalize these memory blocks, we employ a two-stage curriculum. First, we ground them by predicting explicit reasoning steps after each memory block. Second, we discard this step-level supervision and iteratively refine the final answer after each memory block. Our experiments on reasoning benchmarks show that, across language models of different families and sizes, RiM matches or exceeds existing latent reasoning methods while avoiding the autoregressive generation of thoughts. These results demonstrate that large language models can be trained to use working memory as an effective mechanism for latent reasoning.
